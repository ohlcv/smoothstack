## 功能组件描述

### 1. 页面标题区
- 标题：网格交易系统登录/注册
- 左侧展示系统品牌标识和简短描述

### 2. 登录/注册切换
- 切换标签：登录/注册
- 动态切换表单内容
- 平滑的过渡动画

### 3. 登录表单
- 输入项：
  - 用户名/邮箱输入框
  - 密码输入框
- 辅助链接：
  - 记住我复选框
  - 忘记密码链接
- 登录按钮
- 表单验证提示

### 4. 注册表单
- 输入项：
  - 用户名（必填）
  - 电子邮箱（必填）
  - 密码（必填，需符合复杂度要求）
  - 确认密码（必填）
  - 邀请码（可选）
    - 提供输入框
    - 验证邀请码有效性
    - 若邀请码有效，显示邀请人信息
- 用户协议复选框
- 注册按钮
- 注册成功后引导

## 交互说明

1. **表单切换**
   - 登录/注册标签无刷新切换
   - 保留已输入的部分信息
   - 平滑的表单过渡动画

2. **表单验证**
   - 实时输入验证
   - 显示具体的错误提示
   - 登录/注册按钮状态随验证结果变化

3. **邀请码交互**
   - 输入邀请码后实时校验
   - 显示邀请人信息（如果有效）
   - 无效邀请码提示

4. **密码输入**
   - 密码可见/不可见切换
   - 密码强度实时校验
   - 确认密码实时匹配检查

5. **忘记密码流程**
   - 点击后进入找回密码页面
   - 通过邮箱发送重置链接
   - 重置密码流程

## 状态管理

- 表单类型状态（登录/注册）
- 输入验证状态
- 邀请码验证状态
- 表单提交状态

## 响应式设计考虑

- 桌面端：居中卡片式布局
- 平板端：自适应宽度，保持居中
- 移动端：全屏表单，简化布局

## 特殊考虑

- 安全的表单提交
- 友好的错误提示
- 无障碍设计
- 国际化支持
- 密码安全性提示
- 邀请码机制的灵活性
